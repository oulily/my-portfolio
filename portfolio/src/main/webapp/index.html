MusicQ

MusicQ

MusicQ allows multiple users to queue songs to a group queue, and has a voting/ranking system that determines which songs are most liked and will be played first. A host who creates the room has the ability to share a code that allows other users to join.

View Code

Team Members: Samhita Honnavalli, Lily Ou, Gautam Mundewadi, Jennifer Lo, David Farias-Llerenas, Ori Mizrahi

Technologies: Next.js as React framework, MongoDB database, Spotify Authorization and Authentication, Spotify Web API, Spotify Web Playback SDK, and React add-on packages such as useSWR.

GetchaNotes

GetchaNotes

GetchaNotes is a web-based application that converts images of physical notes (hand-written or otherwise) to a digitized version. Notes are categorized using OCR and NLP technologies and can be filtered by topic, viewed, and downloaded.

View Code

Team Members: Josh Chan, Joymaneet Kaur, Lily Ou, and Jeremy Tajonera

Technologies: Google App Engine, Google Vision API, Google Language API, docx4j library, Bootstrap CSS.

Roommates on Fantastic Levels

Roommates on Fantastic Levels (ROFL)

ROFL helps college students find compatible roommates by allowing them to filter other users based on their preferences. Users can create a profile, like another person's profile, and view mutual likes.
